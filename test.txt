1. Поиск товара
	Нажимаем на иконку поиска в правом верхнем углу
	Вводим полное название товара или его часть (более 3 букв) с пробелами перед названием
Ожидаемый результат: Появление искомого товара, начальные пробелы обрезаются
Фактический результат: Ожидание подтвердилось

2. Добавление товара в "Мои покупки"
 	Нажимаем на иконку поиска в правом верхнем углу
	В появившемся поле указываем название товара (например, "рубашка"), нажимаем "Enter"
	Выбираем нужный вариант
	На появившейся странице товара выбираем цвет и размер
	Нажимаем на кнопку "Добавить в мои покупки"
Ожидаемый результат: Товар добавлен в мои покупки, указывается корректная цена заказа, при нажатии на товар в моих покупках можно вернуться на страницу товара и изменить его параметры дл заказа
Фактический результат: Система уведомляет о добавлении товара (справа открывается колонка с моими покупками, где видно, что товар добавлен) и есть возможность перейти на страницу товара, но не изменить его параметры, а только с целью добавить новый

3. Добавление двух товаров в мои покупки одного размера и одного цвета
	Переходим в "Мои покупки"
	Находим нужный товар и нажимаем на него
	На появившейся странице выбираем такой же цвет товара и такой же размер, какой уже есть в "Моих покупках"
	Нажимаем на кнопку "Добавить в мои покупки"
Ожидаемый результат: Товар добавлен в мои покупки - кол-во товара удвоится
Фактический результат: Ожидание подтвердилось, однако для какого-нибудь другого товара может возникнуть ошибка, о которой уведомляет система - товар в ограниченном кол-ве, и тогда товар такого же размера и цвета не добавится повторно

4. Удаление товара из моих покупок (несколько товаров одного цвета и размера)
	Переходим в "Мои покупки"
	Выбираем нужный товар (например, кол-во этого товара 3)
	Нажимаем кнопку "Удалить"
Ожидаемый результат: Один товар удалится 
Фактический результат: Удаляются все экземпляры этого товара, т.е. все 3 штуки (тогда должен быть "-", чтобы уменьшить кол-во товаров, а не только кнопка "Удалить")
		       Если нажать кнопку "Перейти к покупке", то там уже можно отредактировать кол-во товаров.

5. Добавление одинаковых товаров в список желаний
	Нажимаем на иконку поиска в правом верхнем углу
	Вводим название товара, например, "туфли"
	Выбираем нужный вариант и нажимаем "звездочку" (выбирать цвет и размер не надо)
	"Зведочка" стала черной, при повторном нажатии она становится белой, соответственно товар удаляется из списка желаний
Ожидаемый результат: Невозможность добавления одинаковых товаров в список желаний
Фактический результат: Ожидание подтвердилось

6. Регистрация пользователя с неправильным e-mail
	В правом верхнем углу нажимаем на значок регистрации/авторизации
	На странице, на которую перешли, вводим все необходимые данные, e-mail без @, ставим галочку на "Я не робот" и нажимаем кнопку "Зарегистрироваться"
	Поле e-mail будет в красной рамке
	Вводим корректный e-mail, "Я не робот" уже НЕ появляется
	Нажимаем кнопку "Зарегистрироваться"
Ожидаемый результат: Регистрация прошла успешно
Фактический результат: Система уведомляет об ошибке: "Капча не прошла проверку"

7. Фильтрация товаров
	Выбираем категорию товаров, например, "юбки"
	Слева на колонке параметры поиска выбираем нужное, например, белый цвет и сортировка цен по убыванию
Ожидаемый результат: Представлены юбки белого цвета, первыми показаны самые дорогие
Фактический результат: Ожидание подтвердилось

8. Заказ товара при вводе некорректных данных
	Заходим в "Мои покупки"
	Выбираем товар
	Нажимаем на кнопку "Перейти к покупке"
	Вводим данные с неправильным телефонным номером, например 111
	Оставим поле "Почтовый индекс" пустым
	Нажимаем на кнопку "Продолжить"
Ожидаемый результат: Поле для ввода "Почтового индекса" будет выделено, а также номер телефона.
Фактический результат: Ожидание подтвердилось

9. Ввод несуществующего промокода, для получения скидки
	Перейдем в "Мои покупки"
	Введем промокод 5050
	Нажмем кнопку "Применить"
Ожидаемый результат: Скидка 90%
Фактический результат: Система уведомляет об ошибке - неверный код

10. Проверка переходов по внутренним ссылкам
	Перейдем по ссылке "Новинки"
	Перейдем по ссылке "Для женщин"
	Перейдем по ссылке "Для мужчин"
	Перейдем по ссылке "Культовые модели"
	Перейдем по ссылке "Мои покупки"
	Перейдем по ссылке "Подарки"
	Перейдем по ссылке "Нужна помощь?"	
	Перейдем по ссылке "Услуги бутика"
	Перейдем по ссылке "Возвраты"
	Нажмем на логотип
Ожидаемый результат: Все ссылки корректны, отсутствуют неработающие ссылки, логотип является ссылкой на главную страницу
Фактический результат: Ожидание подтвердилось
